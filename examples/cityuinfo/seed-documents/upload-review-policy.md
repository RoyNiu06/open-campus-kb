# Upload Review Policy Example

Uploaded files should not enter the searchable knowledge base automatically.

## Review Checklist

- Is the source clear?
- Is the applicable year clear?
- Does the file contain private personal information?
- Is the document allowed to be summarized or cited?
- Should the content be marked official, student organization official, or student notes?

## Admin Actions

- Approve and ingest
- Reject
- Edit metadata and approve
- Archive
- Delete
